# Library Management System

Welcome to the Library Management System! This project aims to streamline the management of libraries, making it easier for librarians to handle various tasks such as cataloging books, managing memberships, and tracking borrowing and returning of books.

## Features

- **User Management**: Register and manage librarian and member accounts.
- **Book Management**: Add, update, and delete books from the library catalog.
- **Membership Management**: Issue and renew library memberships.
- **Borrowing and Returning**: Keep track of books borrowed by members and manage returns efficiently.
- **Search Functionality**: Easily search for books by title, author, genre, or ISBN.
- **Fine Calculation**: Automatically calculate fines for late returns.
- **Reports**: Generate reports on book availability, membership status, and more.

## Installation

1. Clone the repository: `git clone https://github.com/your-username/library-management-system.git`
2. Navigate to the project directory: `cd library-management-system`
3. Compile the Java files: `javac *.java`
4. Run the application: `java Main`
